## 1. 수정 사항

### 1-1. 장외 시간 종목 스코어 조회 (`/score`) – 기존 기능 검증

**결론**: `/score {종목코드}` 명령어는 **이미 정규장 시간 외에도 정상 동작**함.

| 레이어 | 파일 | 장외 가드 유무 |
|---|---|---|
| Telegram Bot | `commands.js::scoreStock` | ❌ 없음 (정상) |
| REST API | `TradingController::scoreStock` | ❌ 없음 (정상) |
| 서비스 | `OvernightScoringService::calcManualScore` | ❌ 없음 (정상) |

**동작 흐름**:
```
/score 005930
  → kiwoom.scoreStock("005930")
  → GET /api/trading/score/005930
  → OvernightScoringService.calcManualScore()
    1차) Redis WS tick 데이터 조회
         → 있으면 WS 기반 전체 점수 계산
    2차) REST API fallback (ka10001)
         → 없으면 현재가·등락률로 점수 계산
         → data_source = "REST"
```

**Telegram 봇 응답 예시 (장외 REST 모드)**:
```
🟠 삼성전자 (005930) Score Analysis

📊 Total Score: 40pt (Grade B)
❌ 오버나잇 기준(65점) 미달

── Live Data ──
Price: 68,000₩
Change: +0.5%
Exec strength: N/A (WS not subscribed)
Bid ratio: N/A (WS not subscribed)

⚠️ WS not subscribed – score based on price change only
Run /wsStart then retry for full accuracy
```

→ `/help` 메시지에 "💡 /score works anytime (even outside trading hours)" 안내 문구 추가.

### 2-2. 전술 S10 – 52주 신고가 돌파 (`S10_NEW_HIGH`)

**실행 시간**: 11:00~14:00, 15분마다
**사전 필터**: PriceSurgeService (가격급등 종목, 최대 20개)
**API**: ka10081 (주식일봉차트)

**진입 조건**:
| 조건 | 값 |
|---|---|
| 당일 고가 ≥ 52주 최고가 × 0.999 | 신고가 돌파 (0.1% 이내 근접 허용) |
| 양봉 (종가 > 시가) | 음봉 제외 |
| 등락률 | 0.5%~15% |
| 거래량 | 20일 평균 대비 1.5배 이상 |

**점수 공식**:
```
score = 등락률 × 2 + 거래량비율 × 3 + (체결강도-100) × 0.2 + 신고가보너스(20)
```

**매매 파라미터**:
- 진입: 시장가 돌파매수
- 목표 1: +4% / 목표 2: +7%
- 손절: -3%

**오버나잇 적합도**: 30점 (기관·외인 다음으로 높음 – 모멘텀 지속성)

**Telegram 수동 실행**: `/strategy s10`

---

### 2-3. 전술 S12 – 종가 강도 매수 (`S12_CLOSING`)

**실행 시간**: 14:30~15:10, 5분마다
**데이터 소스**: Redis 실시간 WS 틱 데이터

**진입 조건**:
| 조건 | 값 |
|---|---|
| 등락률 | 0.5%~5% |
| 체결강도 (최근 5개 평균) | ≥ 120 |
| 호가 매수/매도 비율 | > 1.5 |

**점수 공식**:
```
score = 등락률 × 3 + (체결강도-100) × 0.3 + 호가비율 × 5
```

**매매 파라미터**:
- 진입: 종가 시장가
- 목표 1: +2% (익일 갭상승) / 목표 2: +4%
- 손절: -2%

**전략 배경**: 장마감 전 기관·외인의 종가 매집 패턴을 포착하여 익일 갭상승 기대.
Redis WS 실시간 데이터만 활용하므로 API 호출 없음 → Rate Limit 무관.

**오버나잇 적합도**: 30점

**Telegram 수동 실행**: `/strategy s12`
