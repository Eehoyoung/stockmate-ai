# StockMate AI – 텔레그램 메시지 예시

> 이 문서는 시스템이 **사용자 명령어 없이 자동으로** 발송하는 메시지 유형과 예시를 정리한다.
> 출처: `telegram-bot/src/handlers/signals.js`, `utils/formatter.js`

---

## 자동 발송 메시지 유형 목록

| 유형 | 트리거 | 발송 주체 | 시간대 |
|------|--------|----------|--------|
| [1] 매수 신호 (ENTER) | AI 분석 결과 ENTER + 점수 ≥ 65 | ai-engine | 장중 상시 |
| [2] 관망 신호 (HOLD) | AI 분석 결과 HOLD + 점수 ≥ 80 | ai-engine | 장중 상시 |
| [3] 강제청산 알림 | 장마감 30분 전 ForceCloseScheduler | api-orchestrator | 14:30 |
| [4] 매매 중단 컨펌 요청 | 뉴스 분석 → PAUSE 권고 | news_scheduler.py | 30분 주기 |
| [5] 뉴스 기반 매매 제어 변경 | CONTINUE / CAUTIOUS 전환 | news_scheduler.py | 30분 주기 |
| [6] 섹터 과열 경보 | 동일 섹터 시간당 신호 ≥ 임계값 | api-orchestrator | 신호 처리 시 |
| [7] 경제 캘린더 사전 경보 | HIGH 영향 이벤트 2시간 전 | api-orchestrator | 이벤트 2h 전 |
| [8] 일일 종합 리포트 | 장 마감 후 자동 생성 | api-orchestrator | 15:30 |
| [9] 봇 시작 알림 | 봇 기동 시 | telegram-bot | 재시작 시 |

## [4] 매매 중단 컨펌 요청 – 자동 발송 (인라인 버튼 포함)

**트리거**: `news_scheduler.py`가 30분 주기로 뉴스 분석 → 부정 심리 감지

```
⚠️ [매매 중단 권고]

AI 뉴스 분석 결과 매매 중단이 권고되었습니다.
시장 심리: 약세 📉

리스크 요인:
• 미국 기준금리 인상 가능성 급등
• 반도체 수출 규제 확대 우려
• 코스피 외국인 대규모 매도 지속

요약: 연준 매파 발언 및 지정학적 리스크 상승으로
      단기 하락 압력이 높은 상황입니다.

매매를 중단하시겠습니까?

[ ✅ 확인 (중단) ]   [ ❌ 취소 ]
```

> 사용자가 **✅ 확인** 클릭 시 → `news:trading_control = PAUSE` 설정
> 사용자가 **❌ 취소** 클릭 시 → 현재 상태 유지

## [5] 뉴스 기반 매매 제어 변경 – 자동 발송

**트리거**: news_scheduler.py가 CONTINUE 또는 CAUTIOUS 전환 감지

### 5-a. 정상 매매 전환

```
✅ [뉴스 기반 매매 제어]
상태: CONTINUE → 정상 매매

시장 심리: 강세 📈
추천 섹터: 반도체, 2차전지, 방산

요약: 경제지표 호조 및 기관 수급 개선으로
      정상 매매 환경이 갖춰졌습니다.
```

### 5-b. 신중 매매 전환

```
⚠️ [뉴스 기반 매매 제어]
상태: CONTINUE → 신중 매매 (CAUTIOUS)

시장 심리: 중립 ↔️
리스크 요인:
• 중국 PMI 예상치 하회
• 원달러 환율 1,400원 돌파

요약: 불확실성 증가. 고위험 포지션은 자제하고
      수급이 확실한 종목 위주로 접근 권장.
```

### 5-c. 수동 제어 변경 알림 (관리자가 명령어 사용 시)

```
🚨 [매매 제어 수동 변경]
CONTINUE → 매매 중단
관리자 명령에 의해 변경되었습니다.
```

## 메시지 흐름 요약

```
┌─────────────────────────────────────────────┐
│           자동 발송 메시지 흐름              │
└─────────────────────────────────────────────┘

[ai-engine / api-orchestrator]
  │
  ├─ 거래 신호 (ENTER/HOLD)   ──→ ai_scored_queue ──→ 텔레그램
  ├─ 강제청산                 ──→ ai_scored_queue ──→ 텔레그램
  ├─ 섹터 과열 경보           ──→ ai_scored_queue ──→ 텔레그램
  └─ 일일 리포트              ──→ ai_scored_queue ──→ 텔레그램

[news_scheduler.py]
  │
  ├─ PAUSE 권고 (컨펌 버튼)   ──→ ai_scored_queue ──→ 텔레그램
  └─ CONTINUE/CAUTIOUS 전환   ──→ ai_scored_queue ──→ 텔레그램

[api-orchestrator EconomicCalendarScheduler]
  └─ 캘린더 사전 경보         ──→ ai_scored_queue ──→ 텔레그램

[telegram-bot index.js]
  └─ 봇 시작 알림             ──→ 직접 전송
```

## 필터 조건 (자동 신호)

신호를 수신했더라도 아래 조건에 해당하면 **사용자에게 발송되지 않음**:

| 조건 | 설명 |
|------|------|
| `ai_score < 65` | ENTER 신호의 최소 점수 미달 |
| `ai_score < 80` | HOLD 신호의 최소 점수 미달 |
| `action == "CANCEL"` | 취소 판단된 신호는 발송 없음 |
| 전략 필터 | 사용자가 `/filter` 로 특정 전략만 구독 설정한 경우 |
| 관심종목 필터 | 사용자가 관심종목(`/watch`) 설정 시 등록 종목만 수신 |
