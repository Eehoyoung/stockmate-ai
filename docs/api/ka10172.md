# [ka10172] 조건검색 요청 일반

사용자가 영웅문4(HTS)에서 설정한 조건식에 해당하는 종목 리스트를 1회성으로 조회합니다.

## 1. 기본 정보
- **Method:** POST
- **URL:** `https://api.kiwoom.com/api/dostk/cond`
- **Content-Type:** `application/json;charset=UTF-8`

## 2. Request 명세
### Body
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| cond_nm | 조건식명 | Y | String | HTS에 저장된 조건명 (예: "급등주포착") |
| cond_id | 조건식ID | Y | String | ka10171(목록조회)에서 얻은 조건식 ID |

## 3. Response 명세 (output)
| Element | 한글명 | 타입 | 설명 |
| :--- | :--- | :--- | :--- |
| stk_cd | 종목코드 | String | 조건에 부합하는 종목코드 리스트 |
| count | 종목수 | String | 검색된 총 종목 수 |
