# [ka10003] 체결정보요청

특정 종목의 당일 시간대별 체결 내역(틱 데이터)을 상세히 조회합니다.

## 1. 기본 정보
- **Method:** POST
- **URL:** `https://api.kiwoom.com/api/dostk/stkinfo`
- **Content-Type:** `application/json;charset=UTF-8`

## 2. Request 명세
### Header
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| api-id | TR명 | Y | String | ka10003 |
| authorization | 접근토큰 | Y | String | Bearer {token} |
| cont-yn | 연속조회여부 | N | String | Y: 이전 데이터 추가 조회 |
| next-key | 연속조회키 | N | String | 응답에서 받은 차기 조회 키 |

### Body
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| stk_cd | 종목코드 | Y | String | 6자리 종목코드 |

## 3. Response 명세 (output)

응답 최상위 배열키: **`cntr_infr`**

| Element | 한글명 | 타입 | 설명 |
| :--- | :--- | :--- | :--- |
| time | 체결시간 | String | HHMMSS 형식 |
| cur_prc | 현재가 | String | 해당 시간의 체결가 |
| flu_prc | 대비 | String | 전일대비 금액 |
| flu_rt | 등락율 | String | 등락율 |
| sell_hoga | 매도호가 | String | 체결 당시 매도 최우선 호가 |
| buy_hoga | 매수호가 | String | 체결 당시 매수 최우선 호가 |
| volume | 체결량 | String | 해당 건의 체결 수량 |
| total_vol | 누적거래량 | String | 당일 누적 거래량 |
| **cntr_str** | **체결강도** | String | **현재 시점의 체결강도 (%)** |

> ⚠️ **주의**: 체결강도 필드명은 `cntr_str` (코드 기준). API 원문 스펙의 `strength`와 다를 수 있음.
> `http_utils.py`의 `fetch_cntr_strength()` 함수에서 `cntr_infr[0].cntr_str` 으로 읽음.

## 4. 전술 사용처

| 전술 | 사용 목적 |
|------|----------|
| S6_THEME_LAGGARD | 테마 후발주 체결강도 확인 (`http_utils.fetch_cntr_strength`) |

## 5. 코드 예시

```python
# http_utils.py
data = resp.json()
cntr_infr = data.get("cntr_infr", [])
latest = cntr_infr[0]
strength = float(latest.get("cntr_str", "0"))
```
