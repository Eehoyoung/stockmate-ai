# ka10019 – 가격급등락요청

> **용도**: 지정 시간 내 급등 종목 스캔 → S4(장대양봉) 보조 스캐너, 모멘텀 종목 조기 포착

**URL**: `POST https://api.kiwoom.com/api/dostk/stkinfo`  
**모의**: `POST https://mockapi.kiwoom.com/api/dostk/stkinfo`  
**Header**: `api-id: ka10019`

---

## Request Body

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `mrkt_tp` | String(3) | Y | `000`전체 `001`코스피 `101`코스닥 `201`코스피200 |
| `flu_tp` | String(1) | Y | **`1`급등 `2`급락** |
| `tm_tp` | String(1) | Y | `1`분전 `2`일전 |
| `tm` | String(2) | Y | 분 또는 일 숫자 입력 (예: `"60"` = 60분 전 대비) |
| `trde_qty_tp` | String(4) | Y | `00000`전체 `00010`만주이상 `00050`5만주 `00100`10만주 `00150`15만주 `00200`20만주 `00300`30만주 `00500`50만주 `01000`백만주 |
| `stk_cnd` | String(1) | Y | `0`전체 `1`관리종목제외 `3`우선주제외 `5`증100제외 `6`증100만 `7`증40만 `8`증30만 |
| `crd_cnd` | String(1) | Y | `0`전체 `9`신용융자전체 |
| `pric_cnd` | String(1) | Y | `0`전체 `1`1천원미만 `2`1천원~2천원 `3`2천원~3천원 `4`5천원~1만원 `5`1만원이상 `8`1천원이상 |
| `updown_incls` | String(1) | Y | `0`상하한 미포함 `1`포함 |
| `stex_tp` | String(1) | Y | `1`KRX `2`NXT `3`통합 |

| 필드 | 설명 | 활용 |
|------|------|------|
| `stk_cd` | 종목코드 | |
| `stk_nm` | 종목명 | |
| `cur_prc` | 현재가 | |
| `flu_rt` | 전일대비 등락률 | |
| **`base_pric`** | **기준가** | **급등 시작 시점 가격** |

## S4 전술 보조 활용

```java
// 30분 내 3% 이상 급등 종목 → S4 장대양봉 후보 우선 스캔
// ka10019 응답 jmp_rt >= 3.0% 종목을 ka10080 분봉 차트로 검증
```

---

## 주의

- `tm_tp=1`(분전) + `tm="30"` → 30분 전 가격 대비 현재 급등률
- `tm_tp=2`(일전) + `tm="1"` → 전일 종가 대비 급등률 (= 일반 등락률)
- `jmp_rt` 와 `flu_rt` 의 차이: `jmp_rt` = 기준시점 대비, `flu_rt` = 전일종가 대비
