# [ka10027] 전일대비등락률상위요청

전일 종가 대비 등락률이 높은 상위 종목 리스트를 조회합니다.

## 1. 기본 정보
- **Method:** POST
- **URL:** `https://api.kiwoom.com/api/dostk/rkinfo`
- **Content-Type:** `application/json;charset=UTF-8`

## 2. Request 명세
### Body
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| mrkt_tp | 시장구분 | Y | String | 000:전체, 001:코스피, 101:코스닥 |
| sort_tp | 정렬구분 | Y | String | 1:상승률, 2:하락률 |
| stk_cnd | 종목조건 | Y | String | 0:전체, 1:관리제외 등 |
| price_low | 가격하한 | N | String | 조회 시작 가격 (공백 시 제한 없음) |
| price_high | 가격상한 | N | String | 조회 종료 가격 (공백 시 제한 없음) |
| vol_cnd | 거래량조건 | N | String | 0:전체, 1:1만주이상 등 |

## 3. Response 명세 (output)
| Element | 한글명 | 타입 | 설명 |
| :--- | :--- | :--- | :--- |
| stk_cd | 종목코드 | String | 종목코드 |
| stk_nm | 종목명 | String | 종목명 |
| cur_prc | 현재가 | String | 현재가 |
| flu_prc | 대비 | String | 전일대비 금액 |
| flu_rt | 등락율 | String | 등락율 |
| vol | 거래량 | String | 현재 거래량 |
| vol_rt | 거래비중 | String | 전일 거래량 대비 비중 |
