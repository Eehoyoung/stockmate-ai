# [ka90002] 테마구성종목요청

특정 테마 그룹에 속해 있는 종목 리스트와 해당 종목들의 시세 정보를 조회합니다.

## 1. 기본 정보
- **Method:** POST
- **URL:** `https://api.kiwoom.com/api/dostk/stkinfo`
- **Content-Type:** `application/json;charset=UTF-8`

## 2. Request 명세
### Header
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| api-id | TR명 | Y | String | ka90002 |
| authorization | 접근토큰 | Y | String | Bearer {token} |

### Body
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| theme_cd | 테마코드 | Y | String | ka90001(테마그룹조회)에서 얻은 테마코드 |

## 3. Response 명세 (output)
| Element | 한글명 | 타입 | 설명 |
| :--- | :--- | :--- | :--- |
| stk_cd | 종목코드 | String | 종목코드 |
| stk_nm | 종목명 | String | 종목명 |
| cur_prc | 현재가 | String | 현재가 |
| flu_prc | 대비 | String | 전일대비 금액 |
| flu_rt | 등락율 | String | 등락율 |
| vol | 거래량 | String | 거래량 |
| market_cap | 시가총액 | String | 해당 종목의 시가총액 |
| theme_nm | 테마명 | String | 소속된 테마 명칭 |
