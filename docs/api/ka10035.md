# [ka10035] 외인연속순매매상위요청

외국인이 일정 기간 동안 연속해서 순매수 또는 순매도한 상위 종목을 조회합니다.

## 1. 기본 정보
- **Method:** POST
- **URL:** `https://api.kiwoom.com/api/dostk/rkinfo`
- **Content-Type:** `application/json;charset=UTF-8`

## 2. Request 명세
### Header
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| api-id | TR명 | Y | String | ka10035 |
| authorization | 접근토큰 | Y | String | Bearer {token} |

### Body
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| mrkt_tp | 시장구분 | Y | String | 000:전체, 001:코스피, 101:코스닥 |
| buy_sell_tp | 매매구분 | Y | String | 1:순매수, 2:순매도 |
| term | 기간 | Y | String | 연속 일수 (예: 3, 5, 10 등) |

## 3. Response 명세 (output)
| Element | 한글명 | 타입 | 설명 |
| :--- | :--- | :--- | :--- |
| stk_cd | 종목코드 | String | 종목코드 |
| stk_nm | 종목명 | String | 종목명 |
| cur_prc | 현재가 | String | 현재가 |
| flu_rt | 등락율 | String | 등락율 |
| vol | 거래량 | String | 거래량 |
| cont_days | 연속일수 | String | 연속 매매 일수 |
| total_qty | 누적합계수량 | String | 해당 기간 누적 순매매 수량 |
