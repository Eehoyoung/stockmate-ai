# [ka10016] 신고저가요청

지정한 기간 내에서 신고가 또는 신저가를 기록한 종목 리스트를 조회합니다.

## 1. 기본 정보
- **Method:** POST
- **URL:** `https://api.kiwoom.com/api/dostk/stkinfo`
- **Content-Type:** `application/json;charset=UTF-8`

## 2. Request 명세
### Header
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| api-id | TR명 | Y | String | ka10016 |
| authorization | 접근토큰 | Y | String | Bearer {token} |
| cont-yn | 연속조회여부 | N | String | Y: 연속조회, N: 일반조회 |
| next-key | 연속조회키 | N | String | 응답에서 받은 next-key 값 |

### Body
| Element | 한글명 | 필수 | 타입 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| mrkt_tp | 시장구분 | Y | String | 000:전체, 001:코스피, 101:코스닥 |
| ntl_tp | 신고저구분 | Y | String | 1:신고가, 2:신저가 |
| high_low_close_tp | 고저종구분 | Y | String | 1:고저기준, 2:종가기준 |
| stk_cnd | 종목조건 | Y | String | 0:전체조회, 1:관리종목제외, 3:우선주제외 등 |
| term | 기간 | Y | String | 일수 입력 (예: 20, 60, 250 등) |

## 3. Response 명세 (output)
| Element | 한글명 | 타입 | 설명 |
| :--- | :--- | :--- | :--- |
| stk_cd | 종목코드 | String | 6자리 종목코드 |
| stk_nm | 종목명 | String | 종목 명칭 |
| cur_prc | 현재가 | String | 현재가 |
| flu_prc | 대비 | String | 전일대비 금액 |
| flu_rt | 등락율 | String | 전일대비 등락율 |
| vol | 거래량 | String | 현재 거래량 |
| high_prc | 최고가 | String | 기간 내 최고가 |
| high_dt | 최고일 | String | 최고가 기록 일자 |
